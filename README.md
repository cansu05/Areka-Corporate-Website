# 🕯️ Areka Carbon Hub - İklim Teknolojisi Platformu

Bireysel ve kurumsal karbon ayak izini ölçen, dengeleme süreçlerini görünür hale getiren ve çok dilli içerik sunan modern bir web uygulaması.

## 🚀 Özellikler

✨ React + Vite tabanlı hızlı arayüz
🌍 Türkçe / İngilizce dil desteği (i18next)
🧭 Çok sayfalı yapı: Anasayfa, Hakkımızda, Manifesto, İletişim
🧾 SEO meta yönetimi ve sayfa bazlı içerik
🎨 MUI tabanlı responsive bileşenler
📬 İletişim formu ve ofis konum bilgileri

## 🛠️ Teknolojiler

- React 19
- TypeScript
- Vite
- Material UI (MUI)
- React Router DOM
- i18next / react-i18next
- ESLint

## 🧩 Proje Yapısı

- `src/pages`: Sayfa bileşenleri
- `src/components`: Ortak UI bileşenleri
- `src/config`: Tema, renkler ve navigasyon
- `public/locales`: Çok dilli çeviri dosyaları

## ⚙️ Kurulum

```bash
npm install
npm run dev
```

## 📦 Komutlar

```bash
npm run dev      # Geliştirme sunucusu
npm run build    # Production build
npm run preview  # Build önizleme
npm run lint     # Lint kontrolü
```
